# GHI CHÚ NỀN TẢNG (đọc trước khi cài)
# =============================================================================
#
# ── WINDOWS 10/11 (x64) ──────────────────────────────────────────────────────
#  1. Cài Visual C++ Build Tools (cho lap, insightface build wheel)
#     https://visualstudio.microsoft.com/visual-cpp-build-tools/
#  2. Cài CUDA Toolkit 12.1 nếu có NVIDIA GPU
#     https://developer.nvidia.com/cuda-12-1-0-download-archive
#  3. Thứ tự cài:
#     pip install torch torchvision --index-url https://download.pytorch.org/whl/cu121
#     pip install -r requirements.txt
#     pip install onnxruntime-gpu  (thay onnxruntime nếu muốn GPU inference)
#
# ── MACOS 13+ INTEL ──────────────────────────────────────────────────────────
#  1. pip install torch torchvision   (wheel CPU+MPS)
#  2. pip install -r requirements.txt
#  NOTE: faiss-cpu wheel hoạt động tốt. insightface cần Xcode CLT:
#        xcode-select --install
#
# ── MACOS 14+ APPLE SILICON (M1/M2/M3/M4) ───────────────────────────────────
#  1. Dùng miniforge/conda (arm64) thay vì Homebrew Python:
#     https://github.com/conda-forge/miniforge
#  2. conda create -n cpose python=3.11 && conda activate cpose
#  3. pip install torch torchvision   (MPS backend tự động)
#  4. pip install -r requirements.txt
#  NOTE: eventlet có thể lỗi trên M-series → thay bằng gevent:
#        pip install gevent  và sửa app_factory.py: socketio = SocketIO(app, async_mode='gevent')
#  NOTE: lap wheel arm64 chưa có PyPI → build từ source hoặc dùng lapjv:
#        pip install lapjv   và sửa import tương ứng trong tracking.py
#
# ── LINUX x64 (Ubuntu 20.04 / 22.04) ────────────────────────────────────────
#  1. apt install -y libgl1 libglib2.0-0   (dependency của OpenCV)
#  2. pip install torch torchvision --index-url https://download.pytorch.org/whl/cu121
#  3. pip install -r requirements.txt
#  4. Nếu server headless: thay opencv-python → opencv-python-headless
#
# ── EMBEDDED: NVIDIA JETSON ORIN / XAVIER (JetPack 5.x, L4T arm64) ──────────
#  - PyTorch: KHÔNG dùng PyPI. Tải wheel từ NVIDIA Developer:
#    https://forums.developer.nvidia.com/t/pytorch-for-jetson/72048
#  - OpenCV: build từ source với CUDA hoặc dùng gói JetPack:
#    sudo apt install python3-opencv
#  - FAISS: build từ source (faiss-cpu PyPI KHÔNG có arm64 wheel)
#    git clone https://github.com/facebookresearch/faiss && cmake ... -DFAISS_ENABLE_GPU=ON
#  - Các gói còn lại trong requirements.txt cài bình thường qua pip
#  - Khuyến nghị dùng opencv-python-headless, bỏ phần [DEV/TESTING]
#
# ── EMBEDDED: RASPBERRY PI 5 (arm64, Python 3.11) ────────────────────────────
#  - PyTorch CPU: pip install torch torchvision (wheel arm64 có từ PyTorch 2.1)
#  - FAISS: build từ source (không có arm64 wheel trên PyPI)
#  - insightface: có thể lỗi build → dùng yunet_sface.py thay arcface
#  - Bỏ: faiss-gpu, onnxruntime-gpu, eventlet (dùng threading mode)
#  - Flask-SocketIO threading mode: SocketIO(app, async_mode='threading')
#  - Hiệu năng: chỉ nên chạy cpose.pipeline.multicam_recorder (Phase 1)
#    Pose + ReID nên offload lên server mạnh hơn
