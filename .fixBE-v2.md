⚠️ LỖI 6: CORS Middleware Chặn WebSocket Handshake
📍 Vị trí: app/config.yaml + app_factory.py
Config:
yamlserver:
  cors_origins: "*"
Code:
pythonCORS(flask_app, resources={r"/*": {"origins": config.server.cors_origins}})
Vấn đề:

Flask-CORS với wildcard "*" không hỗ trợ credentials
Socket.IO client gửi với credentials: true (mặc định)
CORS preflight fail → WebSocket upgrade bị chặn

Hậu quả:

Browser console: Access-Control-Allow-Origin: * không thể dùng với credentials
Socket disconnect liên tục

Bước 4: Fix Recorder Snapshot
python# app/services/recorder.py

class RecorderManager:
    def get_snapshot(self, cam_id: str) -> bytes | None:
        cam_id = str(cam_id).zfill(2)
        with self._lock:
            worker = self._workers.get(cam_id)

        if not worker:
            logger.warning(f"No worker found for cam_id: {cam_id}")
            return None
        
        snapshot = worker.get_snapshot()
        if not snapshot:
            logger.debug(f"Worker {cam_id} returned no snapshot")
        
        return snapshot

function toggleMulticamPreview() {
    if (!state.system.resourcesLoaded && !state.system.folderLoaded) {
        showToast('Chưa có nguồn multicam', 'error');
        return;
    }

    const img = $('#slotSnap1'), empty = $('#slotEmpty1');
    const btn = $('#toggleRtspPreviewBtn'), toggleBtn = $('#multicamToggleBtn');
    
    if (state.preview.multicamVisible) {
        state.preview.multicamVisible = false;
        clearInterval(state.polling.multicam);
        state.polling.multicam = null;
        img.classList.add('hidden');
        empty.classList.remove('hidden');
        btn.innerText = 'Show';
        toggleBtn.innerHTML = '<i class="fas fa-eye-slash"></i>';
        toggleBtn.className = 'cam-toggle off';
    } else {
        state.preview.multicamVisible = true;
        empty.classList.add('hidden');
        img.classList.remove('hidden');
        btn.innerText = 'Hide';
        toggleBtn.innerHTML = '<i class="fas fa-eye"></i>';
        toggleBtn.className = 'cam-toggle on';
        
        const camId = $('#slotSel1').value;
        
        // ✅ FIX: Kiểm tra mode để chọn đúng API
        if (state.system.mode === 'rtsp' && camId) {
            // RTSP mode - cần start recording trước
            state.polling.multicam = setInterval(() => {
                img.src = `/api/cameras/${camId}/snapshot?t=${Date.now()}`;
                img.onerror = () => {
                    console.log('Snapshot failed, camera may not be recording');
                };
            }, 500);
        } else if (state.system.mode === 'multicam_folder' && state.clips.length) {
            // Folder mode - hiển thị clip đầu tiên
            const clip = state.clips[0];
            if (clip?.url) {
                img.src = clip.url;
            }
        }
    }
}
