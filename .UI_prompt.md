# Prompt cho Claude — sửa giao diện và logic CPose dashboard

Bạn đang sửa **trực tiếp code hiện có**, không được refactor lớn, không đổi kiến trúc 3 cột, không thêm framework frontend mới. Giữ nguyên stack hiện tại: `index.html + style.css + app.js + registration.js + Flask backend hiện có`.

## Mục tiêu tổng quát

Giữ nguyên **3 cột chính** của layout hiện tại:

- cột trái: control + status/event
- cột giữa: multicam / registration / original / processed / log kỹ thuật
- cột phải: video clips + live preview

Nhưng chỉnh lại UI/UX và luồng xử lý theo yêu cầu dưới đây.

## A. Yêu cầu bắt buộc

### 1) Cột trái

- Giữ phần header `CPose`, phần status tổng quan, và các nút điều khiển.
- **Thu gọn nút START và STOP**: nhỏ hơn hiện tại, gọn hơn, ít padding hơn, cùng hàng, cân đối hơn.
- **Panel phía dưới chữ “Status”** hiện đang là bảng tóm tắt camera. Hãy đổi panel này thành **bảng sự kiện mini** gồm:
  - cột `TIME`
  - cột `EVENT`
  - cột `CAM`
- Bảng này phải có:
  - kẻ bảng rõ ràng
  - header cố định
  - ô tìm kiếm ở phía trên
  - nút `Export CSV`
- Đây là bảng event tóm tắt ở cột trái, khác với bảng log kỹ thuật ở khung đen giữa phía dưới.

### 2) Khung đen ở giữa phía dưới

- Không còn là “System Logs” chung chung nữa.
- Đổi thành **bảng log kỹ thuật của camera/pipeline**.
- Các cột tối thiểu:
  - `TIME`
  - `CAM`
  - `FPS`
  - `FRAME`
  - `CONF`
  - `ADL`
  - `EVENT`
- Có thể thêm cột `MODE` hoặc `SOURCE` nếu cần, nhưng không bỏ các cột ở trên.
- Dữ liệu ở đây phải lấy từ socket event hoặc runtime event, không phải mock tĩnh.
- Có bộ lọc tìm kiếm riêng cho khung đen này.

### 3) Original View và Processed View

- Mỗi card phải có **1 nút toggle preview** ở header:
  - trạng thái 1: `Show Camera`
  - trạng thái 2: `Hide Camera`
- Khi `Hide Camera`:
  - chỉ **tắt render preview trên UI**
  - **không được dừng backend processing**
  - không được stop pipeline ngầm
  - mục tiêu là giảm buffer/frame render trên trình duyệt
- Khi `Show Camera` lại thì preview tiếp tục hiện.

### 4) Multicam View

- Header của Multicam View phải có đủ các control sau:
  - chọn camera RTSP / hoặc select source nếu cần
  - nút `Load resources.txt`
  - nút `Show/Hide` preview RTSP
  - nút `Upload Folder`
  - đèn trạng thái `REC`
- Luồng đúng:
  1. `Load resources.txt` → upload file `.txt` → gửi backend qua endpoint config hiện có → parse camera → populate danh sách camera/RTSP.
  2. Sau khi đã load thành công, có thể bấm nút `Show` để hiện preview RTSP thời gian thực.
  3. Bấm lại nút đó thì `Hide`, tức là ẩn preview nhưng **không xóa config camera**.
- Multicam View phải hỗ trợ **hiển thị preview RTSP thời gian thực**.
- Nếu phát hiện người bằng **model detect người nhẹ**, backend cần lưu **short video clip** và push clip đó xuống `Original View`.
- Vẫn giữ nút `Upload Folder` để nạp bộ video short quay sẵn từ thư mục multicam.
- Khi folder multicam được nạp thành công, **đèn REC phải chuyển từ đỏ sang xanh lá**.
- Sau khi folder/resource đã sẵn sàng, người dùng bấm `START` thì bắt đầu pipeline pose + ADL.
- Kết quả sinh ra phải:
  - vào cột `Video Clips`
  - ghi log xuống bảng đen giữa phía dưới
  - preview clip tương ứng khi user chọn

### 5) Registration Cam

- Nút `Open` phải là **toggle webcam**:
  - lần 1 bấm: mở webcam, nút màu xanh lá / text có thể là `Open` hoặc `Opened`
  - lần 2 bấm: tắt webcam, nút chuyển màu đỏ / text `Close` hoặc `Closed`
- Khi tắt webcam phải stop toàn bộ media tracks, không để camera còn chạy ngầm.
- Nút `Register`:
  - mở luồng đăng ký người dùng
  - form phải có ít nhất: `Tên`, `Tuổi`, `ID`
  - `ID` phải **tạo tự động tăng dần từ nhỏ đến lớn** (không random chuỗi)
  - nếu backend chưa có generator, hãy thêm generator đơn giản ở backend/service
- Khi user xác nhận đăng ký thành công:
  - trên khung webcam/registration snapshot phải có **bounding box**
  - trong bbox hoặc label phải hiện thông tin cơ bản: `ID`, `Tên`, có thể thêm `Tuổi`

### 6) Live Preview bên phải

- `Live Preview` phải tự mở khi user click vào item phù hợp trong `Video Clips` hoặc vào mục đang được xử lý.
- Nếu user chọn clip trong queue, preview phải chuyển đúng video.
- Dòng trạng thái output dưới preview phải cập nhật theo mode đang chạy, không hardcode mãi `output_labels`.

1. handleResourcesUpload()

Hiện tại resourcesInput là input .txt trong HTML nhưng app.js lại đang xử lý nó như video file, đây là lỗi logic cần sửa .

async function handleResourcesUpload(event) {
  const files = Array.from(event.target.files || []);
  if (files.length === 0) {
    toast("Chưa chọn resources.txt", "error");
    return;
  }

  const file = files[0];

  // Guard 1: chỉ nhận .txt
  if (!file.name.toLowerCase().endsWith(".txt")) {
    toast("Chỉ chấp nhận file resources.txt", "error");
    resetFileInput("resourcesInput");
    return;
  }

  // Guard 2: không cho load khi pipeline đang chạy
  if (state.system.running) {
    toast("Không thể thay resources khi pipeline đang chạy", "error");
    resetFileInput("resourcesInput");
    return;
  }

  try {
    setMulticamBusy(true);
    logSummary("Đang nạp resources.txt", "--");

    const formData = new FormData();
    formData.append("file", file);

    const res = await fetch("/api/config/upload", {
      method: "POST",
      body: formData
    });

    if (!res.ok) {
      const err = await safeJson(res);
      throw new Error(err?.error || "Upload resources thất bại");
    }

    const data = await res.json();

    // Chuẩn hóa camera list
    const cameras = Array.isArray(data.cameras) ? data.cameras : [];

    // Guard 3: file parse xong nhưng không có cam nào
    if (cameras.length === 0) {
      state.system.resourcesLoaded = false;
      state.cameras.rtspList = [];
      updateRecLamp("OFF");
      toast("resources.txt không có camera hợp lệ", "error");
      return;
    }

    state.system.resourcesLoaded = true;
    state.system.folderLoaded = false;
    state.system.mode = "rtsp";
    state.cameras.rtspList = cameras;
    state.cameras.activeRtspCam = cameras[0]?.cam_id || null;

    populateRtspSelect(cameras);     // fill slotSel1
    revealMulticamToggleButton();    // hiện nút SHOW/HIDE
    updateRecLamp("READY");          // REC đỏ -> READY
    updateMulticamEmptyText(`Đã nạp ${cameras.length} camera RTSP`);

    logSummary(`Đã nạp resources.txt (${cameras.length} camera)`, "--");
    logRuntime({
      cam: "--",
      fps: "--",
      frame: "--",
      conf: "--",
      adl: "--",
      message: `Loaded resources.txt with ${cameras.length} camera(s)`
    });

    toast("Nạp resources.txt thành công", "success");
  } catch (err) {
    state.system.resourcesLoaded = false;
    state.cameras.rtspList = [];
    updateRecLamp("OFF");
    toast(err.message || "Lỗi nạp resources.txt", "error");
    logSummary(`Lỗi nạp resources.txt: ${err.message}`, "--");
  } finally {
    setMulticamBusy(false);
    resetFileInput("resourcesInput");
  }
}

6. startPipeline()

Hiện startBtn chỉ fetch("/api/pose/start", { method: "POST", body: JSON.stringify({ mode: "stream" }) }) mà không header, không validate, không await, không bắt lỗi

async function startPipeline() {
  // Guard 1: đang chạy rồi thì không start lại
  if (state.system.running) {
    toast("Pipeline đang chạy", "error");
    return;
  }

  // Guard 2: phải có nguồn
  const canUseRtsp = state.system.resourcesLoaded && state.cameras.rtspList.length > 0;
  const canUseFolder = state.system.folderLoaded && state.clips.length > 0;

  if (!canUseRtsp && !canUseFolder) {
    toast("Chưa có nguồn dữ liệu để chạy pipeline", "error");
    return;
  }

  // Xác định mode
  const mode = canUseRtsp ? "rtsp" : "multicam_folder";

  // Payload chuẩn
  const payload = {
    mode,
    cameras: mode === "rtsp" ? state.cameras.rtspList.map(c => c.cam_id) : [],
    folder: mode === "multicam_folder" ? state.multicamFolderPath || null : null,
    save_overlay: true
  };

  try {
    setMainButtonsBusy(true);
    updateMainStatus("SYSTEM STARTING");
    logSummary(`Bắt đầu pipeline (${mode})`, "--");

    const res = await fetch("/api/pose/start", {
      method: "POST",
      headers: { "Content-Type": "application/json" },
      body: JSON.stringify(payload)
    });

    const data = await safeJson(res);
    if (!res.ok) {
      throw new Error(data?.error || "Start pipeline thất bại");
    }

    state.system.running = true;
    state.system.mode = mode;
    state.pipeline.currentClipId = null;
    state.pipeline.currentCam = null;
    updateRecLamp("RUNNING");
    updateMainStatus("SYSTEM RUNNING");

    disableButton("startBtn", true);
    disableButton("stopBtn", false);

    // Nếu preview đang bật thì giữ polling chạy
    if (state.preview.originalVisible) restartPreviewPolling("original");
    if (state.preview.processedVisible) restartPreviewPolling("processed");
    if (state.preview.multicamVisible) restartPreviewPolling("multicam");

    logRuntime({
      cam: "--",
      fps: "--",
      frame: "--",
      conf: "--",
      adl: "--",
      message: `Pipeline started in ${mode} mode`
    });

    toast("Bắt đầu xử lý thành công", "success");
  } catch (err) {
    state.system.running = false;
    updateRecLamp("READY");
    updateMainStatus("SYSTEM STANDBY");
    toast(err.message || "Không thể start pipeline", "error");
    logSummary(`Lỗi start pipeline: ${err.message}`, "--");
  } finally {
    setMainButtonsBusy(false);
  }
}

function exportSummaryCsv() {
  const query = ($("summarySearchInput")?.value || "").trim().toLowerCase();

  const rows = state.logs.filter(row => {
    const haystack = `${row.time} ${row.event} ${row.cam}`.toLowerCase();
    return haystack.includes(query);
  });

  if (rows.length === 0) {
    toast("Không có dữ liệu để export", "error");
    return;
  }

  const headers = ["TIME", "EVENT", "CAM"];
  const csvLines = [headers.join(",")];

  for (const row of rows) {
    csvLines.push([
      csvEscape(row.time),
      csvEscape(row.event),
      csvEscape(row.cam)
    ].join(","));
  }

  const csvText = "\uFEFF" + csvLines.join("\n"); // BOM để mở Excel tiếng Việt đẹp hơn
  const blob = new Blob([csvText], { type: "text/csv;charset=utf-8;" });
  const url = URL.createObjectURL(blob);

  const a = document.createElement("a");
  a.href = url;
  a.download = `status_log_${buildTimestampForFile()}.csv`;
  document.body.appendChild(a);
  a.click();
  a.remove();

  URL.revokeObjectURL(url);

  toast("Đã export CSV", "success");
  logSummary("Export CSV summary log", "--");
}

async function handleMulticamFolderUpload(event) {
  const files = Array.from(event.target.files || []);

  // Guard 1: không có file
  if (files.length === 0) {
    toast("Chưa chọn folder multicam", "error");
    return;
  }

  // Guard 2: không cho nạp folder khi pipeline đang chạy
  if (state.system.running) {
    toast("Không thể đổi folder khi pipeline đang chạy", "error");
    resetFileInput("multicamInput");
    return;
  }

  try {
    setMulticamBusy(true);
    logSummary("Đang nạp folder multicam", "--");

    // Lọc video hợp lệ
    const videoFiles = files.filter(file => {
      const name = file.name.toLowerCase();
      return name.endsWith(".mp4") || name.endsWith(".avi") || name.endsWith(".mov");
    });

    // Guard 3: folder không có video
    if (videoFiles.length === 0) {
      throw new Error("Folder không chứa video hợp lệ");
    }

    // Parse tên file đúng chuẩn camXX_YYYY-MM-DD_HH-mm-ss.mp4
    const parsed = [];
    const rejected = [];

    for (const file of videoFiles) {
      const meta = tryParseMulticamName(file.name);
      if (!meta) {
        rejected.push(file.name);
        continue;
      }

      parsed.push({
        id: cryptoRandomId(),
        file,
        name: file.name,
        url: URL.createObjectURL(file),
        cam: meta.camId,
        camIndex: meta.camIndex,
        clipDt: meta.clipDt,
        status: "ready",
        progress: 0,
        sourceType: "multicam_file",
        currentFrame: 0,
        totalFrames: 0,
        fps: 0,
        conf: 0,
        adl: "unknown"
      });
    }

    // Guard 4: tất cả file đều sai format
    if (parsed.length === 0) {
      throw new Error("Không có file nào đúng format multicam");
    }

    // Sort time-first, camera-second
    parsed.sort((a, b) => {
      if (a.clipDt.getTime() !== b.clipDt.getTime()) {
        return a.clipDt.getTime() - b.clipDt.getTime();
      }
      return a.camIndex - b.camIndex;
    });

    // Nếu đã có clip cũ -> revoke URL cũ trước khi thay
    cleanupClipObjectUrls(state.clips);

    state.clips = parsed;
    state.system.folderLoaded = true;
    state.system.resourcesLoaded = false;
    state.system.mode = "multicam_folder";
    state.multicamFolderPath = extractCommonFolderPath(files);
    state.pipeline.currentClipId = parsed[0]?.id || null;

    updateRecLamp("READY");
    renderClipQueue();
    revealMulticamToggleButton();
    updateMulticamEmptyText(`Đã nạp ${parsed.length} clip multicam`);

    logSummary(`Đã nạp folder multicam (${parsed.length} clip)`, "--");

    if (rejected.length > 0) {
      logSummary(`Bỏ qua ${rejected.length} file sai format`, "--");
    }

    toast(`Nạp thành công ${parsed.length} clip multicam`, "success");
  } catch (err) {
    state.system.folderLoaded = false;
    state.clips = [];
    updateRecLamp("OFF");
    toast(err.message || "Lỗi nạp folder multicam", "error");
    logSummary(`Lỗi nạp folder multicam: ${err.message}`, "--");
  } finally {
    setMulticamBusy(false);
    resetFileInput("multicamInput");
  }
}

function tryParseMulticamName(fileName) {
  // chấp nhận dạng cam01_2026-01-29_16-26-25.mp4
  const stem = fileName.replace(/\.[^.]+$/, "");
  const parts = stem.split("_");
  if (parts.length < 3) return null;

  const camId = parts[0].toLowerCase();
  if (!/^cam\d{2}$/i.test(camId)) return null;

  const dateStr = parts[1];
  const timeStr = parts[2];
  if (!/^\d{4}-\d{2}-\d{2}$/.test(dateStr)) return null;
  if (!/^\d{2}-\d{2}-\d{2}$/.test(timeStr)) return null;

  const [yyyy, mm, dd] = dateStr.split("-").map(Number);
  const [hh, mi, ss] = timeStr.split("-").map(Number);

  const clipDt = new Date(yyyy, mm - 1, dd, hh, mi, ss);
  if (Number.isNaN(clipDt.getTime())) return null;

  return {
    camId,
    camIndex: Number(camId.replace("cam", "")),
    clipDt
  };
}

function selectClipPreview(clipId, options = {}) {
  const {
    autoplay = true,
    setLivePanel = true
  } = options;

  const clip = state.clips.find(c => c.id === clipId);
  if (!clip) {
    toast("Không tìm thấy clip", "error");
    return;
  }

  const video = $("previewVideoLocal");
  const empty = $("previewVideoEmpty");
  const outputDir = $("outputDirLabel");

  if (!video || !empty) return;

  try {
    // Dừng video cũ
    video.pause();
    video.removeAttribute("src");
    video.load();

    // Chọn source preview
    // Ưu tiên preview backend nếu clip đã xử lý xong và có previewUrl chính thức
    const src = clip.previewUrl || clip.url;
    if (!src) {
      throw new Error("Clip chưa có nguồn preview");
    }

    video.src = src;
    empty.style.display = "none";

    if (autoplay) {
      video.play().catch(() => {});
    }

    if (setLivePanel) {
      state.preview.livePreviewVisible = true;
      state.preview.activePreviewSource = `clip:${clip.id}`;
    }

    if (outputDir) {
      outputDir.textContent = clip.status === "done"
        ? "Out: output_pose"
        : clip.status === "running"
        ? "Out: processing..."
        : "Out: input preview";
    }

    renderClipQueue();
    logSummary(`Chọn preview clip: ${clip.name}`, clip.cam || "--");
  } catch (err) {
    toast(err.message || "Không thể mở preview clip", "error");
    logSummary(`Lỗi preview clip: ${err.message}`, clip.cam || "--");
  }
}

async function handleRegisterSubmit() {
  const name = $("regUserName")?.value?.trim() || "";
  const ageRaw = $("regUserAge")?.value?.trim() || "";
  const source = document.querySelector('input[name="reg_source"]:checked')?.value || "local";
  const rtspUrl = $("regRtspSelect")?.value || "";

  // Guard 1: webcam/source phải sẵn sàng
  if (source === "local" && !state.cameras.registrationCamOpen) {
    toast("Hãy mở webcam trước khi đăng ký", "error");
    return;
  }

  // Guard 2: validate tên
  if (!name) {
    toast("Vui lòng nhập tên", "error");
    focusIfExists("regUserName");
    return;
  }

  // Guard 3: validate tuổi
  const age = Number(ageRaw);
  if (!Number.isFinite(age) || age <= 0 || age > 120) {
    toast("Tuổi không hợp lệ", "error");
    focusIfExists("regUserAge");
    return;
  }

  // Guard 4: không tạo nhiều session song song
  if (state.registration.isCapturing || state.registration.sessionId) {
    toast("Đang có phiên đăng ký chạy", "error");
    return;
  }

  const payload = {
    source,
    rtsp_url: source === "rtsp" ? rtspUrl : null,
    name,
    age
  };

  try {
    setRegistrationBusy(true);
    logSummary(`Đang tạo phiên đăng ký cho ${name}`, "--");

    const res = await fetch("/api/registration/start", {
      method: "POST",
      headers: { "Content-Type": "application/json" },
      body: JSON.stringify(payload)
    });

    const data = await safeJson(res);
    if (!res.ok) {
      throw new Error(data?.error || "Không thể bắt đầu đăng ký");
    }

    // Kỳ vọng backend trả session_id và user_id tăng dần
    const sessionId = data.session_id;
    const userId = data.user_id ?? state.registration.nextUserId;

    if (!sessionId) {
      throw new Error("Server không trả session_id");
    }

    state.registration.sessionId = sessionId;
    state.registration.isCapturing = true;
    state.registration.currentUser = {
      id: userId,
      name,
      age
    };

    // Nếu backend không trả user_id thì fallback tăng cục bộ
    if (data.user_id == null) {
      state.registration.nextUserId += 1;
    } else {
      state.registration.nextUserId = Math.max(state.registration.nextUserId, Number(userId) + 1);
    }

    // Mở modal capture, reset progress UI
    openRegistrationCaptureModal();
    updateRegistrationTitle(`ID: ${userId} | ${name} | ${age}`);
    resetRegistrationProgress();
    startRegistrationSnapshotLoop(sessionId);

    logSummary(`Bắt đầu đăng ký ID ${userId}`, "--");
    toast("Bắt đầu đăng ký thành công", "success");
  } catch (err) {
    state.registration.sessionId = null;
    state.registration.isCapturing = false;
    state.registration.currentUser = null;

    toast(err.message || "Lỗi đăng ký người dùng", "error");
    logSummary(`Lỗi đăng ký: ${err.message}`, "--");
  } finally {
    setRegistrationBusy(false);
  }
}

function syncSocketEventsToUiState() {
  if (!window.io) return;

  socket = io();
  window.appSocket = socket;

  socket.on("connect", () => {
    logSummary("Socket connected", "--");
  });

  socket.on("disconnect", () => {
    logSummary("Socket disconnected", "--");
    updateMainStatus("SOCKET DISCONNECTED");
  });

  // 1. Trạng thái recorder / rec lamp
  socket.on("rec_status", (msg) => {
    const isRecording = !!msg.is_recording;

    state.system.recEnabled = isRecording;
    setText("recordingState", isRecording ? "ON" : "OFF");

    if (isRecording) {
      updateRecLamp("RUNNING");
    } else if (state.system.resourcesLoaded || state.system.folderLoaded) {
      updateRecLamp("READY");
    } else {
      updateRecLamp("OFF");
    }

    logRuntime({
      cam: msg.cam_id || "--",
      fps: msg.fps ?? "--",
      frame: msg.frame ?? "--",
      conf: msg.conf ?? "--",
      adl: "--",
      message: isRecording ? "Recorder active" : "Recorder idle"
    });
  });

  // 2. Camera status realtime
  socket.on("camera_status", (msg) => {
    const camId = msg.cam_id || "--";

    if (msg.fps != null) {
      updateCameraSummary(camId, msg);
    }

    // Nếu Multicam preview đang bật và đúng cam đang chọn thì refresh nguồn snapshot
    if (state.preview.multicamVisible && state.cameras.activeRtspCam === camId) {
      touchPreviewHeartbeat("multicam");
    }

    logRuntime({
      cam: camId,
      fps: msg.fps ?? "--",
      frame: msg.frame ?? "--",
      conf: msg.conf ?? "--",
      adl: "--",
      message: msg.status || "camera_status"
    });
  });

  // 3. Khi có clip mới từ RTSP detect người
  socket.on("clip_saved", (msg) => {
    const clipId = cryptoRandomId();

    state.clips.unshift({
      id: clipId,
      name: msg.filename,
      cam: msg.cam_id || "UNK",
      url: msg.url || null,          // nếu backend có url
      previewUrl: msg.preview_url || null,
      status: "ready",
      progress: 0,
      sourceType: "rtsp_short",
      currentFrame: 0,
      totalFrames: msg.total_frames || 0,
      fps: msg.fps || 0,
      conf: msg.conf || 0,
      adl: "unknown"
    });

    renderClipQueue();

    // Nếu original view đang bật thì ưu tiên hiển thị clip mới nhất
    if (state.preview.originalVisible) {
      state.pipeline.currentClipId = clipId;
    }

    logSummary(`Lưu short clip: ${msg.filename}`, msg.cam_id || "--");
    logRuntime({
      cam: msg.cam_id || "--",
      fps: msg.fps ?? "--",
      frame: "--",
      conf: msg.conf ?? "--",
      adl: "--",
      message: `Short clip saved: ${msg.filename}`
    });
  });

  // 4. Progress pipeline pose/adl
  socket.on("pose_progress", (msg) => {
    state.system.running = true;
    state.pipeline.currentClipId = msg.clip_id || state.pipeline.currentClipId;
    state.pipeline.currentCam = msg.cam_id || state.pipeline.currentCam;
    state.pipeline.currentFrame = msg.frame_id ?? state.pipeline.currentFrame;
    state.pipeline.totalFrames = msg.total_frames ?? state.pipeline.totalFrames;
    state.pipeline.fps = msg.fps ?? state.pipeline.fps;
    state.pipeline.conf = msg.conf ?? state.pipeline.conf;
    state.pipeline.adl = msg.adl || state.pipeline.adl;

    // Đồng bộ clip hiện tại vào queue
    const clip = findClipByRuntimeHint(msg);
    if (clip) {
      clip.status = "running";
      clip.progress = msg.pct ?? clip.progress;
      clip.currentFrame = msg.frame_id ?? clip.currentFrame;
      clip.totalFrames = msg.total_frames ?? clip.totalFrames;
      clip.fps = msg.fps ?? clip.fps;
      clip.conf = msg.conf ?? clip.conf;
      clip.adl = msg.adl || clip.adl;
    }

    renderClipQueue();

    logRuntime({
      cam: msg.cam_id || "--",
      fps: msg.fps ?? "--",
      frame: formatFrame(msg.frame_id, msg.total_frames),
      conf: msg.conf ?? "--",
      adl: msg.adl || "--",
      message: msg.message || "pose_progress"
    });
  });

  // 5. Progress analysis/offline queue
  socket.on("analysis_progress", (msg) => {
    const clip = state.clips.find(c => c.name === msg.clip || c.name.startsWith(msg.clip));
    if (clip) {
      clip.status = "running";
      clip.progress = msg.pct || 0;
      clip.currentFrame = msg.frame_id || 0;
      clip.totalFrames = msg.total_frames || 0;
    }

    renderClipQueue();

    logRuntime({
      cam: clip?.cam || "--",
      fps: "--",
      frame: formatFrame(msg.frame_id, msg.total_frames),
      conf: "--",
      adl: "--",
      message: `analysis_progress ${msg.clip}`
    });
  });

  // 6. Hoàn tất analysis
  socket.on("analysis_complete", (msg) => {
    markRunningClipsAsDone();
    renderClipQueue();
    logSummary("Phân tích hoàn tất", "--");
    toast("Xử lý phân tích hoàn tất", "success");
  });

  // 7. Registration progress
  socket.on("registration_progress", (data) => {
    if (window.faceRegistration) {
      window.faceRegistration.handleProgress(data);
    }

    logRuntime({
      cam: "REG",
      fps: "--",
      frame: "--",
      conf: data.conf ?? "--",
      adl: "--",
      message: data.instruction || "registration_progress"
    });
  });

  // 8. Registration done
  socket.on("registration_done", (data) => {
    if (window.faceRegistration) {
      window.faceRegistration.handleDone(data);
    }

    if (data.status === "success" && state.registration.currentUser) {
      applyRegistrationOverlay(state.registration.currentUser);
      logSummary(`Đăng ký thành công ID ${state.registration.currentUser.id}`, "REG");
    } else {
      logSummary("Đăng ký thất bại", "REG");
    }

    state.registration.isCapturing = false;
  });

  // 9. Lỗi hệ thống
  socket.on("error", (msg) => {
    toast(msg?.message || "Lỗi hệ thống", "error");

    logRuntime({
      cam: msg?.cam_id || "--",
      fps: "--",
      frame: "--",
      conf: "--",
      adl: "--",
      message: msg?.message || "Socket error"
    });
  });
}

function cleanupClipObjectUrls(clips) {
  if (!Array.isArray(clips)) return;

    }
